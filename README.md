## Initialize for dev update

```
$ python -m virtualenv venv --python=python3.10
$ (venv) pip install pygame 
$ pip freeze > requirements.txt
```

## Getting started
Windows
```
$ python -m virtualenv venv --python=python3.10
$ pip install -r requirements.txt 
$ venv\Scripts\activate
$ (venv) python index.py
```
